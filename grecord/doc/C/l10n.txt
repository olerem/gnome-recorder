1. Start Performance Meter (Applications -> System Tools -> Performance Meter).
2. Increase the size of the Performance Meter window to approximately double the default window
   length and double the default window width.
3. Ensure that the window has focus. If it does not, click on the window to give it focus.
4. Wait until the graph is displayed across the full width of the window before you take the screenshot.
4. Use GIMP to take a screenshot of the window. 
5. In GIMP, save the file to PNG.
